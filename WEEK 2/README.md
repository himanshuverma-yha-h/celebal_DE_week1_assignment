# Week 2 Assignment - SQL-Based Data Analysis
## Objective
Analyze sales data using SQL with filtering, aggregation, sorting, and business queries using the Superstore dataset.

## Dataset
- Sample Superstore Dataset
- Total Records: 9,994
- Total Columns: 21

## Tools Used
- Python
- Pandas
- SQLite
- Jupyter Notebook
- VS Code

## Tasks Performed

### 1. Data Loading
- Loaded the Superstore CSV dataset into a Pandas DataFrame.
- Created an SQLite database and imported the dataset into a SQL table.

### 2. Data Exploration
- Explored dataset structure using schema and sample records.
- Reviewed column names, data types, and dataset dimensions.

### 3. Filtering Using WHERE Clause
- Filtered records based on:
  - Region
  - Category
  - Sales
  - Order Date

### 4. Aggregation Using GROUP BY
- Calculated total sales by category.
- Calculated average sales by category.
- Analyzed quantity sold across categories.

### 5. Sorting and Limiting Results
- Identified top-selling products.
- Identified top-performing categories.

### 6. Business Use Cases
- Monthly sales trend analysis.
- Top customers by sales.
- Duplicate record detection.

### 7. Data Validation
- Validated row counts.
- Checked for missing values.
- Verified duplicate records.
- Reviewed data quality metrics.

## Key Insights
- Sales performance varies across categories and regions.
- A small group of customers contributes significantly to total revenue.
- Monthly sales trends highlight fluctuations in business performance.
- Data quality checks confirmed the dataset is suitable for analysis.
